WAMP：
	PHP集成开发环境(phpstudy、phpnow......)
	PHP+Apache+MySQL

	Apache -- Web容器

	集成环境安装目录下的 www 目录是放置 web 服务器资源的目录，将自己需要部署到 web 容器中的资源（包括html、css、js、图像、php......）放置在 www 目录下。
	127.0.0.1 --- 回环测试IP

	放置在 www 目录下的资源，可以使用 http://127.0.0.1/资源路径名称 或 http://localhost/资源路径名称 进行访问。
	如果需要在局域网中访问，则可以通过 http://IP地址/资源路径名称 来进行访问。
		-- 可能需要配置：httpd.conf --> 将 <Directory "D:/wamp/www"> 下的 
		    Deny from all
		    Allow from 127.0.0.1
		改为
			Allow from all
		修改完成后，重启服务

PHP：
	PHP是服务器端脚本程序

	PHP 文件可包含文本、HTML 标签以及脚本。PHP 文件中的脚本可在服务器上执行。

	文件：
		以 .php 作为后缀

	标记：
		<?php

		?>
	输出：
		echo "字符串"; -- 向响应的文档流中输出

	变量：
		$变量名 = 变量值;

	字符串：
		可以使用 "" 或 '' 包括字符串内容
		如果在 "" 中有变量名，则使用变量值替换，如果 '' 中有变量名，则表示字符串本身。
		如果需要连接字符串，使用 . 来连接。
	
	运算符：

	流程控制语句：
		条件结构
		循环结构

	数组：
		数值数组
			$arr = array(5, 7, 1, 4, 9);
			可通过数字下标访问数组元素，如：$arr[0]
		关联数组
			$arr = array("first"=>5, "second"=>7);
			通过字符串索引访问数组元素，如：$arr["first"]

	json_encode()
		将 php 值转换为 json 字符串
	json_decode()
		将 json 字符串解析为 php 值

	表单：
		$_GET 变量用于收集来自 method="get" 的表单中的值
		$_POST 变量用于收集来自 method="post" 的表单中的值。
		$_REQUEST 变量可用于收集 cookie 或 通过 GET 、 POST 方法发送的表单数据的结果。
